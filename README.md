# **Opis**

   <a href="https://play.google.com/store/apps/developer?id=anddev404"><img width="150" align="right" style="margin:0px 0px 0px 0px" src="https://github.com/anddev404/MovieGuide/blob/master/images/icon_512%20new.png?raw=true)](https://play.google.com/store/apps/developer?id=anddev404"></a>
Dzięki **Movie Guide** odkryjesz niesamowite filmy, seriale czy zerkniesz na gwiazdy filmowe. Przejrzysz zdjęcia i wyszukasz informacje o wszystkim co związane z filmowym światem.

Aplikacja składająca się z kilku activity i fragmentów, napisana w języku **Java**.
